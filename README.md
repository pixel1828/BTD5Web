# Bloons TD 5 Web
Bloons Tower Defense 5 was released in 2011 by Kaiparasoft (Now known as Ninja Kiwi), a small studio based in New Zealand, coming from a long franchise, Bloons. The original Bloons was uploaded to Addicting Games, and flooded with mods, such as More Bloons, Even More Bloons, Bloons Player Pack 1, and Bloons Insanity. After these Bloons fan-games, Kaiparasoft made an official spin-off: the first Bloons TD. 

Bloons TD 5 was originally released to mobile, but later came to PC in the form of a Flash game. The mobile and console versions got frequent updates, but the PC version stopped updating not long after. The other versions still update today. The flash version was tucked away, and the actual .swf file was not easy to find.

# The leak

In 2013, Bloons TD 5 was leaked to Newgrounds without any word about it. Using their embed system, I managed to get the file. So, Bloons TD 5 is now a web game. The itch.io upload of this project contains builds for Windows, Mac, and Linux, as well as a simple play-in-browser window.

# Anti-ruffle

Using inspect, I discovered hidden styling in ruffle embeds, including CSS that exposed the tags of things such as the ruffle logo. Changing these to invisible allows it to instantly start the game without a play button or splash screen. This makes it better for easy gaming.
